A tool to work with geo data

What is geo data?

Use cases:

  • locations (aka points)
  • areas

What is geo data?

Use cases:

  • view geo data
  • visualize attached data
  • "paint" new or edit geo data
  • filter data

(Business enviroment)

What are common issue with geo data?

  • data format
  • confidentiality

Problem: we had no tool for this...

Solution: implement a tool

0_o

Requirements:

  • (technical) stand-alone
  • (technical) no manpower available
  • (technical) easy to maintainable
  • (user) use it all day

Technical issues:

  • load data
  • store data

SOLUTION: A stand-alone web application.

  • load data via DnD
  • save data as local download
  • (pro) scriptable via search URL

All data is kept in the browser...
